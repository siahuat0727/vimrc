# vimrc

## Prerequisite

#### Ubuntu

```
sudo apt -y install yarn nodejs npm
```

#### Manjaro

```
sudo pacman -S yarn nodejs npm
```

## Installation

```
curl -L https://raw.githubusercontent.com/siahuat0727/vimrc/master/setup.sh | bash
```
